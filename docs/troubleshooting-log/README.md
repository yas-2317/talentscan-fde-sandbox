# 実践トラブルログ

開発中に実際に起きた問題を、日次の学習内容とは分けて記録する。

## 日次学習ログとの違い

| ログ | 保存する内容 | 保存先 |
|---|---|---|
| 日次学習ログ | その日に理解した概念や仕組み | `docs/learning-log/YYYY-MM-DD.md` |
| 実践トラブルログ | 実際の症状、原因、調査、解決、再発時の手順 | `docs/troubleshooting-log/YYYY-MM-DD-slug.md` |

## 記録する項目

1. 症状
2. 期待する状態
3. 原因
4. 確認した事実
5. 解決方法
6. 再発時の確認手順
7. 関連キーワード

同じ問題が起きたときは、エラーメッセージやキーワードでこのディレクトリを検索し、該当ログを参照する。
